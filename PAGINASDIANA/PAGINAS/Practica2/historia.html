Línea del Tiempo - Evolución de las Aplicaciones Web

1945 - 1970: Origen de Internet

El desarrollo de redes informáticas comenzó con la creación de ARPANET.

1969:

Se establece ARPANET, la primera red de computadoras basada en conmutación de paquetes.

1980 - 1990: TCP/IP y el nacimiento de la Web

El protocolo TCP/IP se adopta como estándar de Internet.

1989:

Tim Berners-Lee propone la World Wide Web (WWW).

1991:

Primer sitio web en línea.

1993 - 2000: Web 1.0 (Sitios Estáticos)

Aparición de los primeros navegadores y páginas HTML estáticas.

1993:

Se lanza Mosaic, el primer navegador gráfico.

1995:

JavaScript y CSS permiten mayor interactividad.

2004 - 2010: Web 2.0 (Interactividad y Redes Sociales)

Se desarrollan aplicaciones dinámicas, redes sociales y contenido generado por usuarios.

2004:

Facebook y YouTube revolucionan la web.

2005:

Uso de AJAX para sitios interactivos.

2010 - Presente: Web 3.0 y 4.0

La inteligencia artificial, blockchain y el metaverso definen la nueva era web.

2014:

HTML5 es adoptado como estándar.

2020 en adelante:

Expansión de la Web Inteligente con IA, asistentes virtuales y computación en la nube.
